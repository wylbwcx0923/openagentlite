# 天气查询技能
## 基础信息
- 唯一标识(skillId)：天气查询
- 归属分类：数据查询
- 版本：v1.0.0
- 作者：项目团队
- 描述：查询指定城市的天气信息

## 功能说明
该技能用于查询指定城市的当前天气信息，包括温度、湿度、风力等数据。适用于用户需要了解某个城市天气状况的场景，解决用户获取天气信息的需求。
请组织调用参数。action应该为**operate**，使用**NetworkAbility**的能力，需要你自己找到一个请求天气的URL，然后调用这个URL，并返回结果。

## 输入参数
| 参数名 | 类型 | 是否必填 | 描述 | 示例 |
|--------|------|----------|------|------|
| city | string | 是 | 城市名称 | 北京 |
| unit | string | 否 | 温度单位（celsius/fahrenheit） | celsius |

## 输出结果
- 成功：返回天气信息，包括温度、湿度、风力等
- 失败：错误码、错误信息、处理建议

## 执行流程
1. 接收输入参数
2. 参数校验
3. 调用网络请求能力获取天气数据
4. 处理返回结果
5. 格式化输出

## 使用示例
### 调用指令示例
查询北京的天气

### 参数示例
{
  "city": "北京",
  "unit": "celsius"
}

### 返回结果示例
{
  "code": 200,
  "message": "执行成功",
  "data": {
    "city": "北京",
    "temperature": "25°C",
    "humidity": "60%",
    "wind": "西北风 3级",
    "condition": "晴"
  }
}

## 注意事项
- 执行权限：需要网络访问权限
- 超时限制：30秒
- 异常处理：网络异常时返回错误信息
- 安全规范：不存储用户查询历史
